# 🏥 Proyecto de Gestión de Enfermería

## 📋 Propósito
Este proyecto es una aplicación web desarrollada con **Symfony** y **Doctrine ORM** que permite gestionar **enfermeros, pacientes y turnos**.  
Incluye un **CRUD completo** (crear, leer, actualizar y eliminar) para los recursos principales y **pruebas automatizadas** con **PHPUnit**, integradas mediante **GitHub Actions** para la **Integración Continua (CI)**.

---

## ⚙️ Instalación

### 🧾 Requisitos previos
- **PHP >= 8.3** (recomendado 8.4)
- **Composer**
- **MySQL / MariaDB** o **SQLite**
- **Symfony CLI** (opcional pero recomendable)
- **Git**
